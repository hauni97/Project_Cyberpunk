## 5. Cost Optimization Strategies

### 5.1 Hardware
- Buy used/refurbished smartphones (saves 50-70%)
- Use existing WiFi adapters if compatible
- 3D print custom cases yourself (saves ~50 EUR)

### 5.2 Software
- Use local LLM instead of API (saves 20 EUR/month)
- Offline STT/TTS engines (no recurring costs)
- Community editions of commercial tools

### 5.3 Time
- Start with Phase 1 only (40% time reduction)
- Use existing scripts/tools where possible
- Join community for shared development
